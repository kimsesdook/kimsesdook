## 👨‍💻 About Me

**"비즈니스 로직의 깊이를 이해하고, 데이터 기반으로 성능을 최적화하는 개발자입니다."**

* 🌱 **현재 주력 분야:** Java, Spring Boot 생태계를 기반으로 한 **Domain-Driven Layered Architecture** 설계 및 API 개발
* 🛠️ **주요 관심사:** 
  * N+1 문제, 복잡한 JPA 쿼리 등 **병목 현상 원인 파악 및 성능 튜닝**
  * Redis를 활용한 데이터 캐싱 및 실시간 데이터 처리
  * 직관적이고 확장 가능한 시스템 아키텍처 구축
* 🔭 **현재 활동:** 직장인 웰빙 관리 HR-SaaS 플랫폼 **`CalmDesk`** 백엔드 고도화 및 **`ClassLink ERP`** 프로젝트 진행 중
